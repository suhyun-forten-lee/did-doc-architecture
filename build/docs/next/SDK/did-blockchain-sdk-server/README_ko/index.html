Server Blockchain SDK

라이브러리​

라이브러리는 Releases에서 찾을 수 있습니다.

Blockchain SDK​

  1. 프로젝트의 libs에 did-datamodel-sdk-server-1.0.0.jar 파일을 복사합니다.
  2. 프로젝트의 build.gradle에 아래 의존성을 추가합니다.
    implementation files('libs/did-datamodel-sdk-server-1.0.0.jar')
    implementation('org.hyperledger.fabric:fabric-gateway-java:2.2.9')
    implementation('com.fasterxml.jackson.core:jackson-databind:2.15.2')
    implementation('org.apache.commons:commons-pool2:2.12.0')
    annotationProcessor('com.fasterxml.jackson.core:jackson-databind:2.15.2')
    annotationProcessor('org.projectlombok:lombok:1.18.28')
    compileOnly('org.projectlombok:lombok:1.18.28')
  1. Gradle을 동기화하여 의존성이 제대로 추가되었는지 확인합니다.
